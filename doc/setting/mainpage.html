各種メンバ( Variable / Function )名の命名規則について

クラス

クラスのメンバはすべて大文字から始めます( アッパーキャメル )。
Intrface クラスに関しては、prefixとして"I"( 英大文字のアイ )を付け、その後、名称を大文字から始めます。
Example : 

class ExampleClass : public IExampleInterface
{
    public void onClick()
    {
        return;
    }
}

メンバ変数

メンバ変数は、以下の規則にしたがって命名されます。

  • 通常変数は prefix に"m"を付け、その後大文字から始める。( prefix は小文字、名称はアッパーキャメル ) 
    int mValiable = 100;
  • static 変数は prefix に"s"を付け、その後大文字から始める。( prefix は小文字、名称はアッパーキャメル ) 
    static int sStaticVal = 3;
  • 定数はすべて大文字とする。 
    static const int DEFINE_VALIABLE = 3;

関数

関数は先頭小文字、以降の意味区切りは大文字のローワーキャメル方式で命名されます。
この規則は、メンバ / グローバル問わず有効です。
また、メソッド引数は prefix に"a"を付け、その後大文字から始めます。( prefix は小文字、名称はアッパーキャメル )

Example : 

    void setNumber( int aNumber )
    {
        vector<int> array;
        try
        {
            mNumber = aNumber;
            array.push_back( aNumber );
        }
        catch( const std::exception& ex )
        {
            Log.e( TAGS, Log.printStackTrace( ex ) );
        }
    }

演算子

ただし、以下の場合には、スペースが入らない場合があります。

  • 単項演算子 
    ++a; a++; !a; ~a;
Example : 
    bool setValue( int aCount, int aNum )

        bool ret = true;

また、括弧同士が隣り合う場合にもスペースは入れません。
